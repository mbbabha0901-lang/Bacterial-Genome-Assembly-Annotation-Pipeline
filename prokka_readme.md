# PROKKA Annotation of *Lactiplantibacillus plantarum*

## Overview

This repository contains genome annotation results generated using Prokka from a de novo assembled genome of *Lactiplantibacillus plantarum*.

The genome assembly was obtained using SPAdes and subsequently annotated using Prokka v1.15.6.

---

## Software Used

| Software | Version |
|----------|---------|
| Prokka | v1.15.6 |
| BLAST+ | v2.16.0 |
| SPAdes | v4.2.0 |
| QUAST | v5.3.0 |

---

## Workflow

Raw Reads

⬇

SPAdes Assembly

⬇

contigs.fasta

⬇

QUAST Assessment

⬇

Prokka Annotation

⬇

Annotated Genome

---

## Input File

- contigs.fasta

---

## Prokka Command

```bash
prokka contigs.fasta \
--outdir genannotat_lp \
--prefix lactiplanti \
--genus Lactiplantibacillus \
--species plantarum
```

---

## Output Files

| File | Description |
|------|-------------|
| lactiplanti.gff | Genome annotation in GFF3 format |
| lactiplanti.gbk | GenBank annotation file |
| lactiplanti.faa | Predicted protein sequences |
| lactiplanti.ffn | Gene nucleotide sequences |
| lactiplanti.fna | Genome nucleotide sequences |
| lactiplanti.fsa | Genome sequence in FASTA format |
| lactiplanti.tsv | Annotation summary |
| lactiplanti.txt | Annotation statistics |
| lactiplanti.log | Log file generated by Prokka |
